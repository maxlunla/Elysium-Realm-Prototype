FOR URP RENDER PIPELINE, PLEASE INSTALL THE URP PACKAGE.

FOR HDRP RENDER PIPELINE, PLEASE INSTALL THE HDRP PACKAGE.

Please follow these steps to switch asset content with URP/HDRP Content:
1. First install the render pipeline from package manager and create/assign the necessary pipeline assets.
2. Once the render pipeline has been switched and in use, close unity & reopen. Now go to this asset's URP/HDRP package and install it.
3. Open/Reopen the demo scene inside the asset and check if everything looks good, you are done.
4. If the materials doesn't look right, try deleting lil toon & the asset folder> import again & repeat the above steps carefully.
